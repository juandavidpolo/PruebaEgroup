-Para la visualizacipón del proyecto es necesario el uso 
del servidor apache para una version de PHP 7 y MySQL (Recomendación XAMPP)
-Se puede crear la base de datos con la opción de importar
para seleccionar el archivo y buscarlo en la carpeta con el nombre de 
"database.sql"
-La conexión se realiza con el usuario "root" sin contraseña.
-Para facilidad y ligereza en el proyecto las imagenes son de enlaces externos
por tanto tener en cuenta a la hora de realizar alguna modificación.
-La carpeta tiene los archivos necesarios para su ejecución.
